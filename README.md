# 🐍 Projetos em Python

Repositório contendo meus projetos desenvolvidos durante meus estudos de Python.

Aqui estão reunidos exercícios, desafios, experimentos e pequenos projetos criados ao longo do curso de Programação em Python. O objetivo deste repositório é registrar minha evolução na linguagem, praticar lógica de programação e desenvolver aplicações utilizando diferentes bibliotecas.

# 🎮 Projetos de Jogos (Destaque)

Os projetos abaixo foram desenvolvidos como forma de praticar programação orientada a objetos, eventos, movimentação, colisão, manipulação gráfica e lógica de jogos.

## 🟦 O Quadrado Fugitivo

Um jogo onde o jogador deve perseguir um quadrado que tenta escapar constantemente.

### Conceitos utilizados

- Loops de jogo
- Eventos do teclado
- Movimentação de objetos
- Colisão
- Geração de posições aleatórias
- Controle de velocidade
- Estruturas condicionais

---

## 🎯 Caça ao Alvo

Jogo baseado em reflexo e precisão, onde o jogador precisa acertar alvos antes que desapareçam.

### Conceitos utilizados

- Atualização em tempo real
- Eventos do mouse
- Sistema de pontuação
- Temporizadores
- Geração aleatória

---

## ⬜ Quadrado Saltitante

Projeto focado em física simples e movimentação automática.

### Recursos

- Gravidade
- Quique nas bordas
- Atualização contínua da tela
- Controle de velocidade
